Catalog

Catalog Re-sync

Re-queue all artists and tracks for re-sync from Spotify. Use this to refresh catalog metadata (images, track details) that may have changed since the last sync.

Wipe Catalog

Delete all catalog data (artists, albums, tracks), remove catalog sync outbox events, set all playlists to sync inactive and delete all playlist checks. This operation cannot be undone.

Confirm Catalog Wipe

This will permanently delete all catalog data, outbox events, playlist checks and deactivate all playlist syncs. Type yes to confirm.
