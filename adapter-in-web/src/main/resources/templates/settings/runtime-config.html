Runtime Config

These settings take effect immediately but are transient — they reset to their default values on the next application restart.

Spotify Request Throttling

Minimum interval between outgoing Spotify API requests. Increasing this value slows down catalog sync but reduces the risk of hitting Spotify rate limits. Default: {config.defaultThrottleIntervalSeconds} s.
